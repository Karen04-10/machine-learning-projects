# CarPrice Prediction

## Project Type
Regression (XGBoost Regressor)

## Description
This project predicts car prices based on various vehicle attributes such as car name, engine type, fuel type, and more. The pipeline includes data preprocessing, categorical encoding, feature scaling, model tuning using GridSearchCV, and evaluation using regression metrics.

## Dataset
Car dataset (car.csv) containing features like:
Car Name, Engine type, Fuel type, Aspiration, Drive wheel, Door number... and Price (target variable)

## Skills & Tools
- Pandas, NumPy, Matplotlib (visualization)
- scikit-learn (GridSearchCV, K-Fold Cross Validation...)
- XGBoost Regressor

## Results / Visualizations
- Scatter plot of actual vs predicted car prices
- Error-based color mapping
- Model performance comparison using R² score, MAE
- Cross-validation score distribution
