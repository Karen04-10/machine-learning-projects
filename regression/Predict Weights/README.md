# Weight-Height Prediction

## Project Type
Regression (Multiple Linear, Polynomial, SVR)

## Description
Predicts weight from height and gender using multiple regression approaches. Comparison of models and evaluation using R² score.

## Dataset
Weight-Height dataset with columns: Gender, Height, Weight

## Skills & Tools
- Pandas, NumPy, Matplotlib
- scikit-learn (Linear Regression, Polynomial Regression, SVR)
- Label Encoding, Train-Test Split, Visualization

## How to Run
1. Run the script `1)Multiple_Linear_Regression.ipynb/py, 2)Polynomial_Regression.ipynb/py, 3)SVR.ipynb/py` for regression models
2. Outputs include predicted vs actual plots and R² scores

## Results / Visualizations
- Scatter plots for actual vs predicted weights
- Model comparison by R² score
