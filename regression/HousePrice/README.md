# House Price Prediction

## Project Type
Regression (Linear Regression)

## Description
Predicts house prices based on multiple features including categorical variables. Handles categorical encoding and scaling.

## Dataset
House dataset with features like area, mainroad, basement, airconditioning, furnishingstatus, and price

## Skills & Tools
- Python (Pandas, NumPy, Matplotlib)
- scikit-learn (LinearRegression, OneHotEncoder, ColumnTransformer)
- Data preprocessing and visualization

## How to Run
1. Run `PricePred.ipynb`
2. Outputs scatter plots and actual vs predicted price comparison

## Results / Visualizations
- Scatter plots showing Area vs Price
- Actual vs Predicted Price plot
