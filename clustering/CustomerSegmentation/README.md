# Customer Segmentation

## Project Type
Unsupervised Learning (KMeans)

## Description
Clusters customers based on Annual Income and Spending Score using KMeans. Uses Elbow method to determine optimal clusters.

## Dataset
Customer dataset with columns: CustomerID, Annual Income, Spending Score

## Skills & Tools
- Pandas, NumPy, Matplotlib
- scikit-learn KMeans
- Elbow method for cluster selection
- Cluster visualization

## How to Run
1. Run `CustSeg.ipynb`
2. Visualizes clusters and centroids

## Results / Visualizations
- Scatter plot showing 4 customer clusters
- Cluster centroids highlighted
