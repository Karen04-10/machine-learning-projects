# Iris Clustering (Hierarchical)

## Project Type
Unsupervised Learning (Agglomerative Clustering)

## Description
Clusters Iris flowers using hierarchical clustering. Visualizes clusters with dendrogram and scatter plots.

## Dataset
Iris dataset with features: sepal length, sepal width, petal length, petal width

## Skills & Tools
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- scipy.cluster.hierarchy (dendrogram)
- scikit-learn AgglomerativeClustering

## How to Run
1. Run `iris.ipynb`
2. Dendrogram and cluster scatter plots generated

## Results / Visualizations
- Dendrogram of hierarchical clustering
- 2D scatter plots for clusters
- Pairplot for feature relationships
