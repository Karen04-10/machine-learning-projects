# Customer Segmentation (Hierarchical)

## Project Type
Unsupervised Learning (Agglomerative Clustering)

## Description
This project performs customer segmentation using hierarchical clustering based on Annual Income and Spending Score. A dendrogram is used to determine the optimal number of clusters (3 and 5), followed by visualization of the resulting clusters using scatter plots.

## Dataset
Mall_Customers dataset with features: CustomerID, Gender, Age, Annual Income, Spending Score

## Skills & Tools
- Python (Pandas, NumPy, Matplotlib)
- SciPy (scipy.cluster.hierarchy for dendrogram)
- scikit-learn (AgglomerativeClustering)

## How to Run
1. Open and run Segmentation.ipynb
2. View the generated dendrogram and cluster visualizations

## Results / Visualizations
- Dendrogram of hierarchical clustering
- 2D scatter plots showing customer segments
