# Credit Card Fraud Detection

## Project Type
Classification (K-Nearest Neighbors - KNN)

## Description
This project detects fraudulent credit card transactions using a KNN classifier. The model is trained on transaction data and evaluated using confusion matrix and F1-score because of class imbalance.

## Dataset
Credit Card Fraud Detection dataset (creditcard.csv)
Contains anonymized transaction features and a target column:
Class: 0 (Normal transaction), 1 (Fraudulent transaction)

## Skills & Tools
- Pandas, NumPy, scikit-learn
- Data Preprocessing, StandardScaler, Confusion Matrix, F1-score

## Results / Visualizations
- Confusion matrix for train and test sets
- F1-score evaluation for performance measurement
- Prediction vs actual comparison output
