# Fake News Detection System

## Project Type
Text Classification

##cDescription
A machine learning model that detects whether a news article is Real or Fake based on its text content. The model uses TF-IDF vectorization to convert text into numerical features and a trained classifier to make predictions.

## Dataset
News dataset containing:
- True news (1)
- Fake news (0)

## Skills & Tools
- Pandas
- Scikit-learn
- TF-IDF Vectorization
- Model Selection method

## How to Run
1. Open the notebook (e.g. news.ipynb)
2. Run all preprocessing and training cells

## Results
- Text-based Fake News classification system
- TF-IDF feature extraction
- Binary classification (Real vs Fake)
- Model successfully distinguishes political/news articles based on language patterns
