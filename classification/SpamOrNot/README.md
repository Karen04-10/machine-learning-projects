# Spam Mail Prediction

## Project Type
Classification (Logistic Regression)

## Description
This project detects spam emails using a Logistic Regression classifier. The model is trained on email text data that is converted into numerical features using TF-IDF vectorization. Result is evaluated using metrics such as confusion matrix, accuracy, and F1-score to handle potential class imbalance between spam and non-spam emails.

## Dataset
Email Spam Dataset (emails.csv)
Contains email text and a target label:
0: Not Spam
1: Spam

## Skills & Tools
- Python (Pandas, scikit-learn)
- Text preprocessing, TF-IDF Vectorization, Confusion Matrix, Accuracy Score, F1-score

## How to Run
1. Open and run `spam.ipynb`
2. View spam vs not spam predictions and evaluation results

## Results / Visualizations
- Confusion matrix for training and testing sets
- Accuracy and F1-score evaluation
- Example predictions (spam vs not spam classification)
