# Breast Cancer Classification

## Project Type
Classification (SVM)

## Description
Classifies breast cancer as malignant or benign using SVM with hyperparameter tuning. Evaluates model accuracy and confusion matrix.

## Dataset
Breast Cancer dataset (columns: features, diagnosis)

## Skills & Tools
- Pandas, NumPy, scikit-learn
- Label Encoding, Train-Test Split, StandardScaler
- GridSearchCV for hyperparameter tuning

## How to Run
1. Run `Cancer.ipynb`
2. Outputs include confusion matrix, cross-validation scores, best parameters

## Results / Visualizations
- Confusion matrix and model accuracy
- Best SVM configuration displayed
