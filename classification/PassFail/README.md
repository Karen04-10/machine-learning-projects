# Pass/Fail Prediction

## Project Type
Classification (K-Nearest Neighbors)

## Description
Predicts student exam outcomes using KNN classifier. Evaluates accuracy and confusion matrix.

## Dataset
Exam dataset with student features and results

## Skills & Tools
- Python (Pandas, NumPy)
- scikit-learn (KNeighborsClassifier)
- StandardScaler, Train-Test Split, Confusion Matrix

## How to Run
1. Run `pass_fail.ipynb`
2. Outputs predicted vs actual results and confusion matrix

## Results / Visualizations
- Table comparing predicted vs actual exam scores
- Confusion matrix and accuracy score
