# Market Basket Analysis

## Project Type
Association Rule Mining (Apriori)

## Description
Analyzes transactional data to find association rules between products.
Null or missing values are automatically excluded from transactions to ensure accurate analysis.

## Dataset
Dataset of 5444 transactions and 33 columns (products and other transaction attributes).
Data source: Data.csv

## Skills & Tools
- Python (Pandas, NumPy)
- apyori (Apriori Algorithm)
- Association rule mining, Lift/Confidence analysis

## How to Run
1. Run `market.ipynb`
2. Outputs association rules sorted by lift
