# Customer Churn Prediction

## Project Type
Classification (Artificial Neural Network)

## Description
Predicts customer churn in telecom using ANN with multiple hidden layers.

## Dataset
Telco Customer Churn dataset with features like services, charges, and demographics

## Skills & Tools
- Python (Pandas, NumPy)
- TensorFlow / Keras
- Label encoding, OneHotEncoding, StandardScaler

## How to Run
1. Run `Customer_Churn.ipynb`
2. Outputs predicted churn vs actual and confusion matrix

## Results / Visualizations
- Table comparing predicted vs actual churn
- Accuracy and confusion matrix
