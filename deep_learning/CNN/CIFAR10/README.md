# CIFAR-10 Image Classification

## Project Type
Classification (CNN)

## Description
Classifies images into 10 categories using a Convolutional Neural Network and predicts the image class label.

## Dataset
CIFAR-10 style image dataset (32x32 RGB images, 10 classes)

## Skills & Tools
- Python (Pandas, NumPy)
- TensorFlow / Keras
- ImageDataGenerator, Conv2D, MaxPooling, Dense, Dropout

## How to Run
1. Run the training script
2. Outputs predicted image classes and accuracy metrics

## Results / Visualizations
- Training vs validation accuracy
- Training vs validation loss
- Table of predicted image classes
- Automatic grouping of images by predicted class
